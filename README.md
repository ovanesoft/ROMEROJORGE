# ROMERO JORGE

Landing page para Romero Jorge - Comercializadora inmobiliaria con desarrollos propios.

## Descripción

Sitio web corporativo que presenta las unidades de negocio:
- **Terráneo**: Desarrolladora de edificios en altura
- **Fideitec**: Fideicomiso tecnológico
- **Comercializadora Romero Jorge**: Servicios comerciales integrales

## Características

- Diseño moderno y elegante
- 100% responsive (móviles, tablets, desktop)
- Navegación suave entre secciones
- Formulario de contacto
- Animaciones y efectos hover

## Secciones

1. **INICIO**: Presentación principal con CTAs
2. **QUIÉNES SOMOS**: Información institucional y misión
3. **ÁREAS/UNIDADES DE NEGOCIO**: Detalle de cada unidad
4. **CÓMO TRABAJAMOS**: Proceso comercial paso a paso
5. **SÚMATE**: Formulario para alianzas y colaboraciones

## Tecnologías

- HTML5
- CSS3 (con variables CSS y Flexbox/Grid)
- JavaScript vanilla
- Diseño responsive con media queries

## Instalación

1. Clonar el repositorio:
```bash
git clone https://github.com/[tu-usuario]/ROMEROJORGE.git
```

2. Abrir `index.html` en tu navegador

## Estructura de archivos

```
ROMEROJORGE/
├── index.html
├── styles.css
├── script.js
└── README.md
```

## Personalización

Para personalizar el sitio:

1. **Colores**: Editar las variables CSS en `styles.css` (líneas 8-17)
2. **Contenido**: Modificar los textos en `index.html`
3. **Imágenes**: Agregar imágenes en las secciones correspondientes
4. **Contacto**: Actualizar información de contacto en el footer
